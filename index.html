Tag´s de textos(voltar)

As tag´s de texto são usadas para organizar deter minado conteúdo textualdentro da pagina.

  1. Os "h" ou as heads

    Essa é a tag usada para definir os títulos do html, e ela vai do "h1" ao "h6",
    sendo que o numeral define o nivel de importância do título na página, onde "h1" tem maior importância e "h6" menor importância.
    Obs:Indepêndente dessas importância, essa estilização pode ser alterada no "CSS"

  2. A tag "p"

    Nessa tag é onde definimos os paragrafos de nosso conteúdo. Nesse elemento, por padrão os navegadores adicionam automáticamente uma linha em branco antes de depois de cada elemento "p".

  3. Tag´s usada para destacar algo no texto

    • "Strog": Essa tag é usada para colocar uma ou mais palavras em NEGRITO.
    • "I": Essa tag é usada para colocar uma ou mais palavras em ITÁLIC.
    • "U": Essa tag é usada para colocar uma ou mais palavras em SUBLINHADO.
    • "MARK": Essa tag é usada para definir uma ou mais palavras como uma seção DESTACADA.
    • "SMALL": Essa tag é usada para definir uma ou mais palavras como um texo MENOR.
    • "SUB": Essa tag é usada para definir uma palavra com detalhe SUBSCRITA → H2o.
    • "SUP": Essa tag é usada para definir uma palavra com detalhe SOBRESCRITO → m2.
    • "DEL": Essa tag é usada para definir uma ou mais palavras que foi EXCLUÍDO do documento.
    • "ABBR": A tag define uma abreviação ou um acrônimo, como "HTML", "CSS", "O sr.", "Dr.", "o mais rápido possível", "ATM".CSS
    • "BLOCKQUOTE": Essa tag é usada para definir uma ou mais palavras como uma seção
      CITADA
      .

      • A tag < font> não é suportada no HTML 5, mas para ela funcionar ela precisa dos seus atributos, color, face.

Lista ordenada(voltar)

Representado pela junção das ol a tag define uma lista ordenada,assim ela pode ser numérica ou alfabética.
Ex: Sintaxe
< ol >
< li >item< /li>
< li >item< /li>
< / ol >
Exibição na página
1.item
2.item

Lista não ordenada(voltar)

Representado pela junção das ul a tag define uma lista nãoordenada,assim ela é exibida com marcadores.
Ex: Sintaxe
< ul >
< li >item< /li>
< li >item< /li>
< / ul >
Exibição na página
• item
• item

Link (voltar)

Link ou âncoras, representado pela tag "a", ela define um hiper link para qual você sera redirecionado para uma página externa ou para um determinado conteúdo da sua propria página, de acordo a os atributos adicionados nessa tag.
